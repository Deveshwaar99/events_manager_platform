<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/Deveshwaar99/events_manager_platform
```

<p>2. install dependencies</p>

```
npm install
```

<p>3. Build the app</p>

```
npm run  build
```

<p>4. Start the app</p>

```
npm start
```
